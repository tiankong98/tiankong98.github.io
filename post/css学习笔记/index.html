CSS基础

基本用法

样式

基本结构：

选择器：重点

声明之间用封号隔开

属性和属性值之间用冒号

样式表

内部、外部样式表

代码注释和格式化

继承性

父元素定义的样式会被子元素继承 主要是字体、颜色和文本属性具有继承性。

层叠性

不同样式作用同一个对象时 ，由于声明同一个属性，会发生优先级。

样式表的优先级：

标签选择器：优先级加权值为1；

伪元素或伪对象选择器：优先级加权值为1；

类选择器：优先值加权值为10；

属性选择器：优先级加权值为10；

ID选择器：优先级加权值为100。
